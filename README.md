# 📚 Slides do Curso

Este projeto é uma página web simples desenvolvida com **HTML, CSS e JavaScript**, hospedada no **GitHub Pages**, que organiza os conteúdos das aulas em formato de **cards (slides)**.

Cada card representa um slide/aula e direciona para o material correspondente.

---

## 🚀 Acesso ao projeto

👉 https://lucasrtorquato.github.io/piagem/

---

## 🧠 Objetivo

Facilitar o acesso dos alunos aos conteúdos das aulas de forma:
- Visual
- Organizada
- Intuitiva

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
